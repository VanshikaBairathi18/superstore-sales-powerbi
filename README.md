# Superstore Sales Dashboard — Power BI

## Project Overview
An interactive sales dashboard built in Power BI using the 
Superstore dataset to analyze sales, profit, and customer trends.

## Pages
- Executive Summary — KPI cards, sales trend, category breakdown
- Regional Analysis — Map, segment donut, region matrix
- Product & Customer — Top 10 products, customers, scatter plot

## Skills Demonstrated
- Data cleaning in Power Query
- Data modeling and relationships
- DAX measures (Total Sales, Profit Margin, YoY Growth)
- Interactive visualizations and slicers

## Dataset
Superstore Sales Dataset from Kaggle

## Tools Used
- Power BI Desktop
- DAX
- Power Query
